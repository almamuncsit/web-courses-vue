## Installtion 
##### Clone the repository
```
git clone https://github.com/almamuncsit/web-courses-vue.git
```

#### Go to project folder
```
cd web-courses-vue
```

##### Install node dependency
```
npm install 
```

##### Run Development server

``` 
npm run serve
```
##### Now open localhost on 8000 port

``` 
http://localhost:8080/
```
