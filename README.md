## Installtion 
##### Clone the repository
```
git clone https://github.com/almamuncsit/web-courses.git
```

#### Go to project folder
```
cd web-courses
```

##### Install node dependency
```
npm install 
```

##### Run Development server

``` 
npm run dev
```
##### Now open localhost on 8000 port

``` 
http://localhost:8080/
```
